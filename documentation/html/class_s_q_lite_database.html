SQLiteDatabase::SQLiteDatabase ( string  dbFilePath,
ToolScanner *  toolScanner 
)

Creates/opens a Database Tools given a filepath for the database. Requires a ToolScanner object that will allow for the database to receive Tool ID's from the toolbox.

If a database that is provided in the dbFilePath parameter, a db file will be created. The function creates a SQL style database with one table named TOOLS with the following fields: ID TEXT (primary key) - the id of the tool as a string NAME TEXT - the human readable name of the tool REG_DATE INT - the date the tool was registered as a time_t value PRIORITY INT - the priority of the tool used to display the tool to the user MISS_COUNT INT - the number of times that the tool has been recorded missing LAST_MISS_DATE INT - the last date and time the tool has been recorded missing as a time_t value

Parameters
dbFilePathpath of a Database Tools .db file
toolScannera ToolScanner object for a physical scanner
